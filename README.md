# Nimyth Server
Nimyth Server source code.

## Installation
Github:
```
git clone https://github.com/hanaui-git/nimyth-server
```
NpmJS:
```
npm i xchacha20-js simple-aes-256 shuffle-seed sovrin-did express
```

## Usage
```
node index.js
```

## Note
Before hosting please set the keys in index.js first.
